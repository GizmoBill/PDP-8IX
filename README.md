\# PDP-8 Simulator



A historically accurate PDP-8/I simulator written in C# WinForms, including

1970's MIT Weather Radar modifications. 8 fields of memory, EAE, two terminals

that can simulate either an ASR-38 Teletype or a VT100, RK05 disks, front

panel with fully functioning lights and switches, extensive code analysis

tools. Will run OS/8 from the RK05.



Working but experimental and under development. Documentation not yet

available but coming.



The Weather Radar environment is described in [this long historical post](https://retrocomputingforum.com/t/timesharing-on-the-mit-weather-radar-pdp-8-ix/5569).





\## Build



Open `PDP8.sln` in Visual Studio 2022 and build the `PDP8Simulator` project.



\## Run



The simulator runs as a standalone WinForms application. No external dependencies.
